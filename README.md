# Marketing_Campaign_Analysis

## Project Overview
Dự án phân tích hiệu quả các chiến dịch Marketing dựa trên dữ liệu từ 2,240 khách hàng. Mục tiêu chính là xác định các đặc điểm nhân khẩu học ảnh hưởng đến tỷ lệ chuyển đổi và xây dựng mô hình phân khúc khách hàng (RFM) để tối ưu hóa ngân sách quảng cáo.
